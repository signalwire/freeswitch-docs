## voicemail_domain

string Sets the domain name in the sender email address when [mod\_voicemail](../FreeSWITCH-Explained/Modules/mod_voicemail_6587070.mdx) emails a voicemail message.

**Usage**

From `conf/directory/default/1010.xml`:

```xml
<include>
  <user id="1010" mailbox="1010">
    <params>
      <param name="password" value="password"/>
      <param name="vm-password" value="1010"/>
    </params>
    <variables>
      <variable name="voicemail_domain" value="speakblast.com"/>
    </variables>
  </user>
</include>
```
