
# origination_callee_id_number 

 Created by  Ryan Harris, last modified on 2018.02.07

## origination_callee_id_number

string Set on the inbound leg to control what caller ID number appears in the caller phone's display. Also see [ignore\_display\_updates](../FreeSWITCH-Explained/Dialplan/Variables-Master-List_16352570.mdx#ignore_display_updates) which affect the processing of these variables.

**Usage**

```xml
<action application="bridge" data="{origination_callee_id_name=Reginald,origination_callee_id_number=2332}sofia/gateway/provider/<Reginald's cellphone number>" />
```
