
# RECORD_STEREO 

 Created by  Ryan Harris, last modified by  livem Chan on 2019.07.15

## RECORD_STEREO

boolean Record leg A and leg B streams (i.e. the caller is recorded to the left channel and the receiver is recorded on right channel) into different channel in a stereo file.

**Usage**

```xml
<action application="set" data="RECORD_STEREO=true"/>
```
