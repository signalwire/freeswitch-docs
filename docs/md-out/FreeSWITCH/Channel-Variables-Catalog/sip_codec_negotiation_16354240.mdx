## sip_codec_negotiation

string `sip_codec_negotiation` is basically a channel variable equivalent of inbound-codec-negotiation.

Options:

* `generous`
* `scrooge`
* `greedy`

Before [74a0cfd1e101413a3941c41d04ee01d8df2ae418](https://freeswitch.org/stash/projects/FS/repos/freeswitch/commits/74a0cfd1e101413a3941c41d04ee01d8df2ae418), `sip_codec_negotiation` will always be overwritten by the sofia profile configuration param `sip-codec-negotiation`. As of this revision, the codec negotiation can be changed on a per-call basis.

**Usage**

```xml
<action application="set" data="sip_codec_negotiation=greedy"/>
```
