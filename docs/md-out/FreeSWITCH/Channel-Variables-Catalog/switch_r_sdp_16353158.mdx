
# switch_r\_sdp 

 Created by  Ryan Harris, last modified on 2018.02.07

## switch_r\_sdp

string readonly This variable holds the remote SDP for the current leg/channel.

**Usage**

![(attenzione)](/images/icons/emoticons/warning.svg) Don't add a carriage return after `"set">` or you'll end up writing a variable with a different name, leaving `switch_r_sdp` with the same value.

```xml
<action application="set"><![CDATA[switch_r_sdp=(sdp here)
]]>
</action>
```
