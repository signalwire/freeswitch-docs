## last_matching_digits

string Contains the last set of digits that the user dialed when using the "dmachine" digit-handling. This is most commonly used with the [bind\_digit\_action](../FreeSWITCH-Explained/Dialplan/Variables-Master-List_16352570.mdx#last_bridge_to) dialplan application.
