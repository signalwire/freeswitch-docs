## last_bridge_hangup_cause

string This is set to the hangup cause of the last bridged B leg of the call. For a list of hangup causes, see [Hangup Causes](../FreeSWITCH-Explained/Troubleshooting-Debugging/Hangup-Cause-Code-Table_3964945.mdx#about).

**Usage**

```xml
<action application="log" data="B-leg hangup cause: ${last_bridge_hangup_cause}"/>
```
