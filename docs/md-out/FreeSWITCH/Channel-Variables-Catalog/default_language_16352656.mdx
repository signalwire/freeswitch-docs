## default_language

string Controls the default language the [Say Phrase engine](../FreeSWITCH-Explained/Configuration/Speech-Phrase-Management_9634492.mdx#0-about) will use when no language is explicitly specified in the API call. This permits you to easily support multiple languages by only changing a single variable at call time.

**Usage**

```xml
<action application="set" data="default_language=fr"/>
```
