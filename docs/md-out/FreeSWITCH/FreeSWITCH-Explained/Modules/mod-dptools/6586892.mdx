# Usage

This application may be run inline from the [XML dialplan](../../Dialplan/XML-Dialplan-archive_6586601.mdx#why).

Also, you can set the following in autoload\_configs/switch.conf.xml, &lt;param name="verbose-channel-events" value="yes"/>

fsctl verbose\_events on|off works, too

# Example

```xml
<condition field="destination_number" expression="^Monroy$">
  <action application="verbose_events" data="true"/>
  <action application="answer"/>
</condition>
```
