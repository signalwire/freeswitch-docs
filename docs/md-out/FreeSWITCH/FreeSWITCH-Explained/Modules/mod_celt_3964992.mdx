
# mod_celt 

 Created by  Belaid Areski on 2014.06.13

# About

mod\_celt allows you to have the HD CELT audio codec in FreeSWITCH.

## Setup

Uncomment mod\_celt in src/modules.conf and in conf/autoload\_configs/modules.conf.xml

## Test

Brian has set up a test that plays a 48kHz sound file: sip:886@taz.bkw.org:5080
