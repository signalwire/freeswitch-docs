
# mod_av 

 Created by  Attila Gulyas, last modified by  John Boteler on 2020.10.14

"_FS Video Codec / File Format using [libav.org](http://libav.org)_", quoting from [mod\_av.c](https://github.com/signalwire/freeswitch/blob/master/src/mod/applications/mod%5Fav/mod%5Fav.c) . Enabled by default in [modules.conf.xml for vanilla installations](https://github.com/signalwire/freeswitch/blob/a5cecbfc2a5e0ea2d3f45489f2681a67e32ce955/conf/vanilla/autoload%5Fconfigs/modules.conf.xml).

mod\_av can be used to record and playback rtmp:// streams.
