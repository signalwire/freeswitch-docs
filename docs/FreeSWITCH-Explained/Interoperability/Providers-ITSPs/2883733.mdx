
# Provider Iristel (Canada) 

 

## [IrisTel.com](http://IrisTel.com)

Last update: 17 November 2010

This is a simple configuration to use your iristel broadfone service with FreeSWITCH.

NOTE: Iristel don't accept any caller id number, should by your Canadian number provided by iristel

  
Gateway configuration /conf/sip\_profiles/external/iristel.xml

```xml
<include>
  <gateway name="iristel"> <!-- used in dialstrings -->
    <param name="username" value="your_canadian_number"/>
    <param name="auth-username" value="your_auth_username, no canada number"/> 
    <param name="realm" value="iristel.net"/> <!-- iristel domain name -->
    <param name="from-user" value="your_canada_number"/>
    <param name="from-domain" value="iristel.net"/> <!-- iristel domain for caller id -->
    <param name="proxy" value="sbc1.iristel.net"/> <!-- Iristel SBC -->
    <param name="password" value="secret"/>
    <param name="extension-in-contact" value="true"/> <!-- could by set because iris need contact yournumber@iristel.com
    <param name="expire-seconds" value="120"/>
  </gateway>
</include> 
```
