
# Provider iCall (USA) 

 

## [iCall.com](http://iCall.com)

Last update: 21 October 2010

* Configuration for iCall carrier/wholesale type accounts (not consumer)
* \[[carriers.icall.com](http://carriers.icall.com/)\]
* Works with local inbound DIDs, toll-free inbound, and outbound SIP service
* Place the following in _conf/sip\_profiles/external/icall.xml_:
* Replace USERNAME and PASSWORD with the account info you were provided. Leave the cust\_ as a prefix to the username.
* **NOTE!!** As of June 2009, iCall has directed customers to use [sbc01-car.dal.us.icall.net](http://sbc01-car.dal.us.icall.net) for **outgoing domestic** calls.

   <param name="realm" value="sbc01-car.dal.us.icall.net" />
   <param name="register" value="false" />
 </gateway>
 <gateway name="icall_international">
