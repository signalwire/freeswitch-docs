
# Provider Belcentrale (Netherlands) 

 

## [Belcentrale.nl](http://Belcentrale.nl)

Last update: 17 November 2010

In _conf/directory/default/_ (or default) create a file belcentrale.xml:

```xml
<include>
     <gateway name="belcentrale.nl">
        <param name="username" value="31XXXXXXXXX"/>
        <param name="realm" value="sip1.belcentrale.nl"/>
        <param name="password" value="your_password"/>
        <param name="register" value="true"/>
    </gateway>
</include> 
```

then in sip\_profiles/nat.xml (or default.xml when not using NAT) edit the setting for auth\_calls and set to false:

```xml
<param name="auth-calls" value="false"/>
```

DTMF does not seem to work with RFC2833\. Had to set phone to use in-audio DTMF.
