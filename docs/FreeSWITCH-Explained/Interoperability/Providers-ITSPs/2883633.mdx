
# Provider BBTel (Argentina) 

 

## BBTel.net (Argentina)

Last Update: 18 June 2009

Create _sip\_profiles/external/bbtel.xml_ with:

```xml
<include>
  <gateway name="sip.bbtel.net">
    <param name="username" value="6385xxxx"/>
    <param name="auth-username" value="<PIN HERE>"/>
    <param name="from-user" value="6385xxxx"/
    <param name="from-domain" value="sip.bbtel.net"/>
    <param name="password" value="<PW HERE>"/>
    <param name="extension" value="6385xxxx"/>
    <param name="proxy" value="sip.bbtel.net:5061"/>
    <param name="register-proxy" value="sip.bbtel.net:5061"/>
    <param name="register" value="true"/>
    <!--<param name="extension-in-contact" value="true"/>-->
  </gateway>
</include> 
```
