
# Provider Gafachi (USA) 

 

## [Gafachi.com](http://Gafachi.com)

Last update: 30 September 2010

  
The Authentication ID and Secret are provided inside your account, under Main-Menu->Configuration Information (not the ones you use to login to your account.)

/conf/sip\_profiles/external/gafachi.xml

```xml
 <include>
  <gateway name="sip.gafachi.com">
    <param name="username" value="<Authentication ID>"/>
    <param name="password" value="<Secret>"/>
    <param name="register" value="true"/>
  </gateway>
</include>
```

You may also need to provide the realm and/or proxy (ping [sip.gafachi.com](http://sip.gafachi.com) to find out what IP to use)

```xml
    <param name="realm" value="64.192.112.13"/>
    <param name="proxy" value="64.192.112.13:5060"/> 
```

Gafachi requires that you \*must\* provide a valid caller id with each request, or the call will fail with a 604 error. Something like:

```xml
[effective_caller_id_number=YOUR_NUMBER,effective_caller_id_name=YOUR_NAME]/sofia/gateway/sip.gafachi.com 
```
