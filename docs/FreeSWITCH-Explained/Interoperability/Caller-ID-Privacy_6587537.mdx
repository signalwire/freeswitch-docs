
# Caller ID Privacy 

 

## About

There are various methods to restricting the presentation of Caller\*ID.

## Setting CID Method

In the channel: [Channel\_Variables#Caller\_ID\_Related](https://wiki.freeswitch.org/wiki/Channel%5FVariables#Caller%5FID%5FRelated "Channel Variables"), specifically: [sip\_cid\_type](../../Channel-Variables-Catalog/sip_cid_type_16352846.mdx)

In the gateway: [Sofia Gateway Authentication Params](../Configuration/Sofia-SIP-Stack/Sofia-Gateway-Authentication-Params_7143510.mdx#about)\- you can set "caller-id-in-from" to be true.

## Privacy

[origination\_privacy](../../Channel-Variables-Catalog/origination_privacy_16353327.mdx) \- Sets privacy profile for caller. Options are "screen", "hide\_name", "hide\_number".

\<action application="set" data="origination_privacy=hide_name"/>

Or you can use : as a deliminator

\<action application="set" data="origination_privacy=hide_name:hide_number:screen"/>

* Caller-Privacy-Hide-Name privacy\_hide\_name .
* Caller-Privacy-Hide-Number privacy\_hide\_number This variable tells you if the inbound call is asking for CLIR\[Calling Line ID presentation Restriction\] (either with anonymous method or Privacy:id method)
