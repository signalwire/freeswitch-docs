
# ODBC.connect 

 

## About

How to connect to an ODBC database

## Usage

Connect to an ODBC database.

### Return value

* true - Successfully connected to database
* false - Connect failed, check DSN settings.

### Example

```bash
use("ODBC");
var db = new ODBC("dsnName", "username", "password");
db.connect();
```
