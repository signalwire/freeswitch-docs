
# ODBC.query 

 

## About

 
```xml
var row, result;
result = db.query("sql statement");
while (db.nextRow()) {
   row = db.getData();
   console_log("Row[0]: " + row[0] + "\n");
}
```

### Return value

* bool - True if the statement was successfully invoked.
