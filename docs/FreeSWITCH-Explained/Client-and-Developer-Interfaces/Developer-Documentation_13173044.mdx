## Development

FreeSWITCH maintains a Team City Continuous Integration (CI) server at [build.freeswitch.org](http://build.freeswitch.org):8111\. (Thanks DRK!) Contact [msc@freeswitch.org](mailto:msc@freeswitch.org) if you would like access and wish to set up a build agent on our Team City server.
