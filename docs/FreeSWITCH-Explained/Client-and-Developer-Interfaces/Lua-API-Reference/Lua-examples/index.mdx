
# Lua examples 

 

## About

This page will list all FreeSWITCH Lua examples.

* [Lua Welcome IVR example](./Lua-Welcome-IVR-example_3965157.mdx#about) \- A simple Lua IVR to start with.
* [Lua Intercom example](./Lua-Intercom-example_3965150.mdx#about) \- Calls a defined list of extensions that are not currently active in a call then auto answers the call.
* [Lua MythTV alert example](./Lua-MythTV-alert-example_3965153.mdx#about) \- Example to send alert to MythTV
* [Lua Conference Room Management example](../../../Conference/Lua-Conference-Room-Management-example_3965172.mdx#about) \- Simple conference room management
* [Lua ASR TTS Directory example](./Lua-ASR-TTS-Directory-example_1049011.mdx) \- Guide to configure automatic speech recognition (ASR) to look up users in a company directory
* [Lua Directory example](./Lua-Directory-example_1049009.mdx#about) \- Example of directory in Lua
* [Lua Database agent login example](./Lua-Database-agent-login-example_3965212.mdx#about) \- Simple agent login script using a PostgreSQL database
* [Lua example Bridging two calls with retry](./Lua-example-Bridging-two-calls-with-retry_3965224.mdx#about)
* [Lua Fakecall responder example](./Lua-Fakecall-responder-example_3965229.mdx#about) \- Simulate random call answering and saying things for a random times once answered
* [Lua Group Pickup example](./Lua-Group-Pickup-example_3965238.mdx#about) \- Simulate group pickup as in Asterisk.
* [Lua example Send mail when no answer](./Lua-example-Send-mail-when-no-answer_3965242.mdx#about) \- Test the bridge status and send mail to the destination's owner to notify the missed call.
* [Lua TeleCaptcha example](./Lua-TeleCaptcha-example_3965261.mdx#about) \- script Tele-Captcha, this allows to act as a simple audio captcha to verify human.
