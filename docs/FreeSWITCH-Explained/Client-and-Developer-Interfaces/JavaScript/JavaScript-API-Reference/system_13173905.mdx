
# system 

 

## About

Executes an external command.

Click here to expand Table of Contents

* 1 [Synopsis](#synopsis)
* 2 [Example](#example)

### Synopsis

```xml
var return_code = system( command );
```

Output of the command is not returned. If you need to send or receive output from the command, see the [File](https://wiki.freeswitch.org/wiki/File "File") object.

### Example

```xml
system("create_account");
```
