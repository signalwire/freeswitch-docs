
# JS Flite 

 Created by  John Boteler on 2015.03.13

## About

Example demonstrating how to invoke Flite text-to-speech.

## Example

**Flite example**

```js
 if ( session.ready( ) ) {
    session.answer( );     
    session.execute("set", "tts_engine=flite");
    session.execute("set", "tts_voice=kal");
    session.execute("speak", "This is an example of flite text to speech engine.");
    session.execute("set", "playback_terminators=#");     
    session.hangup( );
 }
```
