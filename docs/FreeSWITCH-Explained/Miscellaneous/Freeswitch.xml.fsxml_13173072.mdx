
# Freeswitch.xml.fsxml 

 

Located under the \{freeswitch\_installation}/log/ directory. This file is the how the FreeSWITCH pre-processor compiles your \{freeswitch\_installation}/conf/\* files together. This file is a great place to start when troubleshooting XML and/or configuration issues.

You could replace your freeswitch.xml file with this one if necessary for testing and troubleshooting.
