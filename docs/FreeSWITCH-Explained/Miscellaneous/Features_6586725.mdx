### FreeSWITCH Features

#### Default implementation is for a PBX or Softswitch

* The core ([libfreeswitch](https://wiki.freeswitch.org/wiki/Libfreeswitch "Libfreeswitch")) can be [embedded](https://wiki.freeswitch.org/wiki/Embedding%5FFreeSWITCH "Embedding FreeSWITCH") into almost any app that can use a .so or .dll.
* Mold it into a soft phone, PBX, soft-switch or anything in between.

#### Modular system allows you to extend the system easily

* Applications may be written in C, [Lua](../Modules/mod_lua_1048944.mdx#--), Java, .NET, [Javascript/ECMAScript](https://wiki.freeswitch.org/wiki/Javascript "Javascript"), Python, [Perl](https://wiki.freeswitch.org/wiki/Perl "Perl"), and more!

#### Handle thousands of concurrent channels with media on a standard PC

See [Scalability Testimonials](https://wiki.freeswitch.org/wiki/Scalability%5FTestimonials "Scalability Testimonials").

#### Interoperates with many different products and protocols

* Such as [Bayonne](http://www.gnutelephony.org/index.php/GNU%5FBayonne), [Yate](http://yate.null.ro/pmwiki/), [sipXecs](http://www.sipfoundry.org/) or [Asterisk](http://www.asterisk.org/).
* Supports SIP, H.323, SCCP, LDAP, Zeroconf, XMPP / Jingle etc.
* With [FreeTDM](https://wiki.freeswitch.org/wiki/FreeTDM "FreeTDM") a BSD licensed TDM abstraction library it can interface with the PSTN as well.

#### Multi-tenant

See [multi-tenant](https://wiki.freeswitch.org/wiki/Multi-tenant "Multi-tenant").

#### Shared Line Appearance ([SLA](https://wiki.freeswitch.org/wiki/SLA "SLA"))

Turn on the following in sip\_profiles/internal.xml:

```xml
<param name="multiple-registrations" value="true"/>
<param name="manage-presence" value="true"/>
```
