
# How To Create a SignalWire Personal Access Token 

 

  
SignalWire Personal Access Tokens (PAT)s are required to access FreeSWITCH install packages. To get an access token, sign up for a SignalWire account at [signalwire.com](https://id.signalwire.com/signup/account/new).

  
1. Log into your SignalWire account
2. Select Profile from the menu in the upper right corner of your space then click Personal Access Tokens.

![](/attachments/67240087/67240092.png)

  
3\. Click "Generate a New Token" and name it. Click "Generate Token"

![](/attachments/67240087/67240091.png)

  
4\. Copy the Personal Access Token to save it. It will not be shown to you again. Click "I've Saved It" when ready.

![](/attachments/67240087/67240090.png)

  
5\. Your token will now show up in the list. It can be deleted by clicking "Revoke".

![](/attachments/67240087/67240089.png)
