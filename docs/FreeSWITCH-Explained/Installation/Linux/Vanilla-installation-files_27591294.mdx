Description of files and folders created during a FreeSWITCH installation (for example following the Debian 9 Stretch installation instructions), to help one decide which files are really needed.

**For the most up-to-date list, see [conf/vanilla](https://github.com/signalwire/freeswitch/tree/master/conf/vanilla) in the [FreeSWITCH Github repo](https://github.com/signalwire/freeswitch).**

Top level entries are named after the [global variable](../../Introduction/Variables/Global-Variables_13173994.mdx#about) for the containing directory (see [Directories](../../Introduction/Variables/Global-Variables_13173994.mdx#directories) section). For example, the main configuration directory on Debian 9:

**\`conf\_dir\` global variable's value in \`fs\_cli\`**

```xml
freeswitch@tr2> eval $${conf_dir}
/etc/freeswitch
```

## 1 `conf_dir` (\~> `/etc/freeswitch` )

### 1.1 `./config.FS0` 

Seems to be a sample fax modem configuration used by [mod\_spandsp](../../Modules/mod_spandsp_6587021.mdx)), although `config.FS0` is not mentioned there, and basing this on a [signalwire/freeswitch search](https://github.com/signalwire/freeswitch/search?q=config.FS0&unscoped%5Fq=config.FS0), yielding the following line:

```xml
 2) copy conf/config.FS0 to /var/spool/hylafax/etc (or wherver the appropriate dir is on your system)
```

### 1.2 `./extensions.conf` 

An example Asterisk dialplan. See [mod\_dialplan\_asterisk](../../Modules/mod_dialplan_asterisk_3966416.mdx#about).

### 1.4 `./freeswitch.xml` 

The main configuration file. See [Configuring FreeSWITCH](../../Configuration/Configuring-FreeSWITCH/index.mdx#set) or just [freeswitch.xml section](../../Configuration/Configuring-FreeSWITCH/index.mdx#freeswitchxml).

## 2\. Directories in conf_dir

### 2.1 `./yaml/` 

Seems to be created for people who complained regarding XML and/or wanted to use Asterisk style configuration. See this [FAQ entry](../../Miscellaneous/FAQ/index.mdx#q-xml-sucks-do-i-have-any-other-options).

Committed by Anthony Minessale II in 2008 with commit message "[Yet Another Mega Labotomy](https://github.com/signalwire/freeswitch/commit/35dc909cf69191d51af26e26adb4d4d5bcbfcdaa)", and it hasn't been touched since. [mod\_yaml](../../Modules/mod_yaml_32178409.mdx#-freeswitch--mod_yaml-) also hasn't been documented since 2008.
