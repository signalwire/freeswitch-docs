
# Working with Sound Files 

 

## About

Working with sound files in a telephony environment can be challenging. Fortunately there are tools, tips, and tricks that can help you out.

## Tools

* [Sox](http://sox.sourceforge.net/) \- SOund eXchange, great for manipulating sound files  
   * Converting WAV files using sox: [audio\_conversion\_cheat\_sheet](https://www.stefaanlippens.net/audio%5Fconversion%5Fcheat%5Fsheet/)
* [Audacity](http://audacity.sourceforge.net/) \- Graphical sound editor and analysis software
* [Ecasound](http://ecasound.seul.org/ecasound/)  \- Multi-track audio processor
