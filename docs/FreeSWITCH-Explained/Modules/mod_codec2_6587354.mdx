
# mod_codec2 

 

## About

codec2 is a narrow band codec created by the developer of the speex codec.

## Instalation

codec2 is included in the freeswitch source tree, just add it in module.conf codecs/mod\_codec2 save, and do: ./configure make make install.

## Usage

Add mod\_codec2 to modules.xml

```xml
<load module="mod_codec2"/>
```

## Configuration

Configuration in autoload_configs/codec2.conf.xml.

```xml
<configuration name="codec2.conf" description="Core Configuration">
    <settings>
        <param name="mode" value="3200" />
    </settings>
</configuration>
```

The mode can be changed to set the bitrate (values: 3200, 2400, 1400 and 1200) in bps, by default "2400" will be used.
