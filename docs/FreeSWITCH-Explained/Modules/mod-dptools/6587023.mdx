
# mod_dptools: info 

 

## About

Dumps channel information to console.

## Usage

```xml
info 
```

```xml
<action application="info"/> 
```

To execute on B-leg:

```xml
 <action application="set" data="bridge_pre_execute_bleg_app=info"/>
```

Example with a different loglevel from a Lua script:

```xml
session1:execute("info","notice");
```
