
# mod_dptools: set_user 

 Created by  Maximiliano Guerra de Medeiros, last modified by  Ryan Harris on 2018.02.08

# About

Pulls all of the channel variables defined for a user as if the user had auth'd.

# Usage

```xml
set_user <user@domain> [prefix]
```

This application may be run [inline](./6586667.mdx#about) from the XML dialplan.

# Example

```xml
<action application="set_user" data="5551212@${domain}"/>Â 
```

Or, to prefix the names of all variabes by for example "dest\_" do:

```xml
<action application="set_user" data="5551212@${domain} dest_"/>
```
