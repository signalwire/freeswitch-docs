
# mod_dptools: log 

 

## About

Logs a string of text to the console. See [mod\_logfile](../mod_logfile_1048990.mdx#about) for more information on logging.

Click here to expand Table of Contents

* 1 [Usage](#usage)
* 2 [Examples](#examples)
* 3 [See Also](#see-also)

### Usage

```bash
log [loglevel] <log message>
```

loglevel defaults to DEBUG if not specified.

This application may be run [inline](../../Dialplan/XML-Dialplan/index.mdx#about) from the XML dialplan.

### Examples

```xml
<action application="log" data="DIALING Extension DialURI [${sip_uri_to_dial}]"/>
<action application="log" data="INFO DIALING Extension DialURI [${sip_uri_to_dial}]"/>
<action application="log" data="DEBUG DIALING Extension DialURI [${sip_uri_to_dial}]"/>
```
