# mod_opus — Opus Codec

`mod_opus` provides the Opus wideband/fullband audio codec.

It is documented in detail in **[Chapter 16: Codecs and Negotiation](../../media/codecs-and-negotiation.mdx)** and **[Appendix B: Codec Reference](../../reference/appendix-codec-table.mdx)**. This page is the module-index entry; see there for its configuration, parameters, and usage in context.
