# mod_event_socket — The Event Socket

`mod_event_socket` exposes the FreeSWITCH event system and API over TCP (the interface `fs_cli` uses).

It is documented in detail in **[Chapter 25: The Event Socket](../../integration/event-socket.mdx)** and **[Chapter 46: The Inbound Event Socket](../../programming/esl-inbound.mdx)**. This page is the module-index entry; see there for its configuration, parameters, and usage in context.
