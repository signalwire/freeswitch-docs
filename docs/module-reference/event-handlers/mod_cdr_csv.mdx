# mod_cdr_csv — CSV Call Detail Records

`mod_cdr_csv` writes one CDR per leg to CSV files using a configurable template.

It is documented in detail in **[Chapter 27: Call Detail Records](../../integration/call-detail-records.mdx)**. This page is the module-index entry; see there for its configuration, parameters, and usage in context.
