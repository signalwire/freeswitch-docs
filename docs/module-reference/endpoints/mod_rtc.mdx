# mod_rtc — WebRTC Transport

`mod_rtc` provides the shared WebRTC DTLS-SRTP transport used by Verto and SIP-over-WSS.

It is documented in detail in **[Chapter 9: WebRTC with Verto](../../users-endpoints/webrtc-verto.mdx)** and **[Chapter 10: WebRTC over SIP (WSS)](../../users-endpoints/webrtc-sip-wss.mdx)**. This page is the module-index entry; see there for its configuration, parameters, and usage in context.
