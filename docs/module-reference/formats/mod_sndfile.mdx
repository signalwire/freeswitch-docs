# mod_sndfile — Audio File Formats (libsndfile)

`mod_sndfile` reads and writes WAV/AIFF/AU and other audio files via libsndfile.

It is documented in detail in **[Chapter 18: Audio Files and Streaming Sources](../../media/audio-files-and-streaming.mdx)**. This page is the module-index entry; see there for its configuration, parameters, and usage in context.
