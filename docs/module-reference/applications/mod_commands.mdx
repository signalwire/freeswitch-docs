# mod_commands — Core API Commands

`mod_commands` registers the core `fs_cli`/ESL API commands such as `originate`, `uuid_*`, `show`, `status`, and `reload`.

It is documented in detail in **[Chapter 31: CLI and API Command Reference](../../reference/cli-and-api.mdx)**. This page is the module-index entry; see there for its configuration, parameters, and usage in context.
