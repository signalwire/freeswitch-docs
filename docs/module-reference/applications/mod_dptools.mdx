# mod_dptools — Core Dialplan Applications

`mod_dptools` registers the standard dialplan application set — `answer`, `bridge`, `playback`, `record`, `set`, `transfer`, and dozens more.

It is documented in detail in **[Chapter 14: Dialplan Application Reference](../../dialplan/dptools-reference.mdx)**. This page is the module-index entry; see there for its configuration, parameters, and usage in context.
