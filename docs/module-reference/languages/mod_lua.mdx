# mod_lua — Lua Scripting

`mod_lua` embeds the Lua runtime for dialplan scripts, API commands, XML handlers, and event hooks.

It is documented in detail in **[Chapter 28: Scripting Integration](../../integration/scripting.mdx)** and **[Chapter 48: Embedded Scripting APIs](../../programming/scripting-apis.mdx)**. This page is the module-index entry; see there for its configuration, parameters, and usage in context.
