# mod_flite — Flite Text-to-Speech

`mod_flite` provides the bundled Flite (Festival-lite) text-to-speech engine. Select it from the dialplan with the `speak` application or by setting `tts_engine=flite`; bundled voices include `kal`, `awb`, `rms`, and `slt`.

**See also:** [mod_tts_commandline](./mod_tts_commandline.mdx) for text-to-speech via an external command.
