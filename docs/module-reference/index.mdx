This part is a per-module reference for the bundled modules that the topical
chapters do not document on their own. Each page treats one module as a
self-contained unit: what it does, the module name to load, its configuration
file and parameters (where one exists), the dialplan applications and API
commands it registers, and the channel variables it reads or sets. Every value
is verified against the module source and the shipped configuration.

Modules are grouped by the `src/mod/` category they belong to. None of these
modules are required for a basic PBX; most are optional and are not in the
default `modules.conf.xml` load set unless noted on the page.
